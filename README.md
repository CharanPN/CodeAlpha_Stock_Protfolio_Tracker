#  Stock Portfolio Tracker

A simple **Python-based stock portfolio tracker** that allows users to simulate buying stocks and calculates the **total investment value**. The program also saves a **portfolio summary** into a text file for later reference.

##  Features

- Displays a list of available stocks and their prices
- Allows users to purchase multiple stocks
- Calculates investment value for each purchase
- Updates quantity when the same stock is purchased again
- Tracks total investment
- Saves portfolio details to a **portfolio.txt** file
- Simple terminal-based interface

##  Concepts Used

This project demonstrates the use of:

- Python Dictionaries
- Loops (`while`)
- Conditional Statements
- User Input Handling
- File Handling
- Basic Arithmetic Operations

##  How the Program Works

1. The program displays the list of available stocks and prices.
2. The user enters a **stock symbol**.
3. The user enters the **quantity of shares**.
4. The program calculates the **investment value**.
5. If the same stock is purchased again, the quantity is updated.
6. The user can add more stocks or exit.
7. A **portfolio summary** is saved to `portfolio.txt`.
